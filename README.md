# 开发环境搭建（仅限 macOS）

## 安装 [Homebrew](https://brew.sh/)

```bash
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## 安装 Python 3

```bash
$ brew install python
```

**注：新版的 Homebrew 运行 `brew install python` 命令会默认安装 Python 3。**

## 安装 [Pipenv](https://docs.pipenv.org/)

```bash
$ brew install pipenv
```
